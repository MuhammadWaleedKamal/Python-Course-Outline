# IF Statements

## Simple IF
```python
if condition:
    # Execute if True
```

## IF-ELSE
```python
if condition:
    # Execute if True
else:
    # Execute if False
```

## IF-ELIF-ELSE
```python
if condition1:
    # Execute if condition1 True
elif condition2:
    # Execute if condition2 True
else:
    # Execute if all False
```

## Nested IF
```python
if condition1:
    if condition2:
        # Both conditions true
```

## Use Cases
- Validate input
- Check eligibility
- Route program logic
- Handle errors
