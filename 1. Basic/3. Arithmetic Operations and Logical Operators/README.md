# Arithmetic Operations and Logical Operators

## Arithmetic Operations

- **+** Addition: 10 + 5 = 15
- **-** Subtraction: 10 - 5 = 5
- **\*** Multiplication: 10 * 5 = 50
- **/** Division: 10 / 5 = 2.0
- **//** Floor division: 10 // 3 = 3
- **%** Modulo: 10 % 3 = 1
- **\*\*** Exponent: 2 ** 3 = 8

## Logical Operators

- **and**: True if both are true
- **or**: True if one is true
- **not**: Reverses boolean

## Precedence
1. Parentheses ()
2. Exponents **
3. *, /, //
4. +, -
